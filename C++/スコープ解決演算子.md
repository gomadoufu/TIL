# スコープ解決演算子

使い方が2通りあると解釈するとわかりやすいかも。

所有者 :: 変数とか

ドット演算子とは別物だと考えた方がよい。(アロー演算子を自然に受け入れるためにも)



 ## グローバル変数を呼び出すときのスコープ演算子

``` c++
#include <stdio.h>

int     x= 100;

int main(void)
{   int x= 200;

    printf("内部の x= %d\n", x);
    printf("外部の x= %d\n", ::x);

    return 0;
}

```



## クラスの外からpublicなメンバを呼び出す時のスコープ演算子

``` C++
class CLAS{ 
  private:
    int val;
  public:
    void set(int);
    int get();
};

void CLAS::set(int v){   
  val = v;
}
int CLAS::get(){   
  return val;
}
```
