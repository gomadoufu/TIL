# パターンマッチを使わない、素朴なgcd解法

gcd1 ... 定義から  
gcd2 ... ユークリッドの互除法  
  

``` haskell 
gcd1 x y = findComDiv x
  where
    findComDiv d =
      if comDiv d then d else findComDiv (d - 1)
    comDiv d = divisible x && divisible y
      where
        divisible x = x `mod` d == 0

gcd2 x y = if y == 0 then x else gcd2 y (x `mod` y)
```
